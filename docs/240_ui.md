# Пользовательский интерфейс

## Экраны для разных ролей

### Клиент
- Регистрация по телефону
- Просмотр баланса бонусов
- История операций

### Бариста (сенсорный интерфейс)
- Категории товаров (Кофе/Чай/Десерты)
- Плитки товаров с ценами
- Корзина с подсчетом сумм
- Поиск клиента по телефону
- Списание бонусов
- Завершение заказа

### Администратор
- Управление меню
- Просмотр заказов
- Статистика продаж
