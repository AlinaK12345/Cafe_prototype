# Архитектура CoffeeLab Loyalty System

## Общая архитектура
- Монолитное Spring Boot приложение
- REST API для взаимодействия
- H2/PostgreSQL база данных

## Схема компонентов
[Клиент] → [Контроллеры] → [Сервисы] → [Репозитории] → [БД]

## Технологический стек
- Java 17
- Spring Boot 3.4.2
- Spring Data JPA
- H2 Database
- Maven
